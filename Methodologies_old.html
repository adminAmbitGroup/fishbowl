Process Story

It starts with a question.

Next we unpack the problem. Relevant details refine and focus our efforts.

Then we comb the source, identifying relevant data to provide an answer.

Using the data, we build a model using the Statistical Analysis System (SAS).

A broad range of analysis expertise provides tangible insights.

Automated reports provide continued monitoring of key performance indicators.
